SKTilemapParser

open class SKTilemapParser: NSObject, XMLParserDelegate

The SKTilemapParser is a custom XMLParserDelegate parser for reading Tiled TMX and tileset TSX files. To read a tile map, used the SKTilemapParser.load method:

if let tilemap = SKTilemapParser().load(fromFile: "sample-file") {
   scene.worldNode.addChild(tilemap)
}
