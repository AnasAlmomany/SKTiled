SKGroupLayer

open class SKGroupLayer: TiledLayerObject

The SKGroupLayer object is a container for grouping other layers.

Add layers to the group with:

groupLayer.addLayer(playerLayer)

Remove with:

groupLayer.removeLayer(playerLayer)
